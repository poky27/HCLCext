☁️ Sincronización con Google Sheets
Conecta tu HCE con una hoja de Google para guardar y acceder a tus pacientes desde cualquier dispositivo.
Pega aquí la URL de despliegue de tu Apps Script
Cómo configurar (5 min):
  1. Ve a script.google.com → Nuevo proyecto
  2. Pega el código Apps Script (botón de abajo)
  3. Crea una Google Sheet → Copia el ID de la URL
  4. Pega el ID en el código (línea: SHEET_ID)
  5. Desplegar → Web App → Ejecutar como: Tú → Acceso: Cualquiera
  6. Copia la URL de despliegue y pégala arriba
